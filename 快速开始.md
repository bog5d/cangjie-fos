# 快速开始 — 5 分钟上手

> 你只需要会"双击"和"复制粘贴"，不需要懂任何代码。

---

## 第一步：解压文件

把收到的压缩包（`.zip` 文件）解压到全英文路径，例如：

```
C:\FOS\CangJie_FOS
```

> ⚠️ **不要解压到带中文的路径**（如"桌面""下载"）— 可能导致启动失败。  
> 正确：`C:\FOS\`   错误：`C:\Users\张三\桌面\`

解压后你会看到一个叫 `CangJie_FOS` 的文件夹。

---

## 第二步：填写 API Key（第一次必做）

**双击这个文件：**

```
填写API密钥_双击我.bat
```

它会自动打开记事本，把 `=` 后面填入王波给你的 Key：

```
SILICONFLOW_API_KEY=这里填Key（必填）
DEEPSEEK_API_KEY=这里填Key（必填）
DASHSCOPE_API_KEY=这里填Key（可选）
```

填完保存，关掉记事本，向导会自动检查格式是否正确。

> 这一步只做一次，以后直接启动就行。

---

## 第三步：双击启动

找到这个文件，双击：

```
一键启动_Python版.bat
```

弹出黑色窗口后：

- **首次启动**：需要下载 Python 依赖（约 250MB），**请耐心等待 5~15 分钟，不要关闭窗口！**
- **之后每次启动**：约 10 秒完成

看到这行字说明成功了：

```
✅ CangJie FOS 已启动
访问: http://localhost:8000
```

浏览器会自动打开，进入系统主页。

> ⚠️ **黑色窗口不要关！** 关了系统就停了。用完了再关。

---

## 第四步：开始用

浏览器打开后，你会看到主页。

**上传路演录音：**
1. 点右上角「复盘上传向导」
2. 填机构名称、选场景、填被访谈人名字
3. 上传录音文件（支持 .m4a .mp3 .mp4 .wav）
4. 点「提交」，等 5-10 分钟
5. 在「任务列表」里看结果

**问豆豆问题：**  
右侧有个对话框，直接打字问，比如：
- "见红杉要准备什么材料？"
- "刚才的报告里最严重的问题是什么？"

---

## 遇到问题怎么办

| 现象 | 解决方法 |
|------|---------|
| 弹出"Windows 保护了你的电脑" | 点「更多信息」→「仍要运行」 |
| 黑色窗口一闪而过 | 右键 → 以管理员身份运行 |
| 首次启动卡住不动 | 正在下载依赖（约250MB），**耐心等10分钟**，不要关窗口 |
| 浏览器没自动打开 | 手动在浏览器地址栏输入 `localhost:8000` |
| 显示"Python 未找到" | 重新安装 Python，记得勾选"Add to PATH" |
| 豆豆说"网络不通" | 先双击`填写API密钥_双击我.bat`检查Key是否填对 |
| 提示"uv 安装失败" | 网络问题，切换手机热点重试 |
| 解压路径有中文导致崩溃 | 把文件夹移动到 `C:\FOS\` 等纯英文路径 |
| 其他问题 | 双击运行`诊断_打不开请运行我.bat`，截图发给王波 |

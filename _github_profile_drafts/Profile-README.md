## 🎯 核心专注 · Focus

**中文**  
一级市场融资是个典型的"人脑密集型"信息战场——机构偏好、路演反馈、尽调清单散落在录音、微信、笔记里，从不被系统化捕捉。我在用 LangGraph + FastAPI + SQLite 构建一套**自主融资状态机**，让隐性知识变得结构化、可检索、可追溯。

**EN**  
Primary market fundraising is an intensely human-memory-driven battlefield — investor preferences, pitch feedback, and DD requests scatter across recordings, WeChat, and notes, never systematically captured. I'm building an **autonomous fundraising state machine** using LangGraph + FastAPI + SQLite to make tacit knowledge structured, searchable, and auditable.

### 🔒 Cangjie FOS (Private) · 仓颉融资作战操作系统

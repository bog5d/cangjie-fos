# 仓颉 FOS — 同事安装上手指南

> 写给：第一次拿到这套系统的同事  
> 前提：你不需要懂代码，只要会双击和复制粘贴  
> 预计时间：首次约 15~30 分钟（主要是等下载）

**若你手里的压缩包根目录里已有 `00_先看这一行.txt`、`仓颉FOS-使用指引（收到压缩包后请读）.md` 和 `点击开始-仓颉FOS.bat`：** 建议**先**按根目录那三样做（尤其《使用指引》里「从收到压缩包到能正常用」整段），再回来看下面逐步说明。两文档内容互补，不重复犟读。

---

## 准备工作：先找王波要这两样东西

在开始之前，确认你手上有：

- [ ] `CangJie_FOS_Release_xxxxxxxx.zip` 压缩包
- [ ] API Key（王波发给你的，一串字母数字，形如 `sk-xxxxxxxx`）

没有的话先找王波要，再继续下面的步骤。

## 第一步：检查 Python 是否已安装

**Python 是这套系统的运行基础，必须先装好。**

### 怎么检查？

1. 按键盘上的 `Win + R`（同时按 Windows 徽标键和 R 键）
2. 弹出"运行"对话框，输入 `cmd`，按回车
3. 黑色窗口打开后，输入下面这行，回车：

   ```
   python --version
   ```

4. 看结果：
   - ✅ **显示 `Python 3.xx.xx`（版本号 3.11 或以上）**：已安装，跳到第二步
   - ❌ **显示"不是内部或外部命令"或版本低于 3.11**：需要安装，继续下面

---

### 如果没有 Python，怎么安装？

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点击黄色大按钮"Download Python 3.x.x"（选最新的 3.12 或 3.13）
3. 下载完成后，**双击运行安装程序**
4. 安装界面会出现，**务必勾选最下方的"Add Python to PATH"**（这一步最关键，不勾会装了也没用）

   ```
   ┌─────────────────────────────────────────────┐
   │  Install Python 3.12                        │
   │                                             │
   │  ○ Install Now                              │
   │  ○ Customize installation                   │
   │                                             │
   │  ☑ Add Python to PATH    ← 必须勾上！       │
   └─────────────────────────────────────────────┘
   ```

5. 点"Install Now"，等安装完成
6. 重新打开 cmd，再次输入 `python --version` 确认显示版本号

## 第二步：解压文件到正确位置

**⚠️ 不能解压到中文路径（如桌面、下载），否则系统可能启动失败。**

### 推荐做法：

1. 在 C 盘根目录新建一个文件夹，命名为 `FOS`
   - 打开"此电脑"→ 双击 C 盘 → 右键空白处 → 新建文件夹 → 输入 `FOS`
2. 把 ZIP 压缩包复制到 `C:\FOS\`
3. 右键压缩包 → 解压到当前文件夹

**重要（与「能打开页面但录音/豆豆报错」强相关）：** 除本压缩包外，你还需要**并列**的 **`AI_Pitch_Coach`** 文件夹（王波会一并提供或从内部仓库拉取），且与 `CangJie_FOS` 在同一父目录下。桥接数据目录 **`.fos_data`** 一般也在该父目录下，由「仓颉资产台账 FSS」向上扫描后生成 `asset_index.json`（无 FSS 时资产台账可为空，属正常）。

解压并放好双仓后，推荐目录结构如下：

```
C:\FOS\
  ├── AI_Pitch_Coach\          ← 必须，且含 src\ 子目录
  │     └── src\               ← 无此目录则豆区/上传会报 Pitch_Coach 错误
  ├── CangJie_FOS\
  │     ├── 一键启动_Python版.bat
  │     ├── 填写API密钥_双击我.bat
  │     ├── 诊断_打不开请运行我.bat
  │     ├── 安装并启动.ps1
  │     ├── 快速开始.md
  │     ├── backend\
  │     └── frontend\
  └── .fos_data\               ← 可选；FSS 扫描后才有 asset_index.json
        └── asset_index.json
```

若把 `AI_Pitch_Coach` 放在别处，在 `CangJie_FOS\backend\.env` 中增加一行：`CANGJIE_PITCH_COACH_ROOT=你的绝对路径`（无引号）。详见 `docs\RELEASE_CHECKLIST.md`。

## 第三步：填写 API Key

**双击这个文件：**

```
填写API密钥_双击我.bat
```

### 你会看到什么？

黑色窗口打开，显示填写说明，然后**记事本自动弹出**，内容大概是这样：

```
# 仓颉 FOS — API Key 配置
SILICONFLOW_API_KEY=
DEEPSEEK_API_KEY=
DASHSCOPE_API_KEY=
KIMI_API_KEY=
```

### 怎么填？

把王波给你的 Key 复制粘贴到等号后面，例如：

```
SILICONFLOW_API_KEY=sk-abcdefg12345678（这里是王波给你的key）
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx（这里是王波给你的key）
```

注意：
- `=` 后面直接粘贴，**不要加引号，不要加空格**
- `DASHSCOPE_API_KEY` 和 `KIMI_API_KEY` 可以先不填

填完后：
1. 按 `Ctrl + S` 保存
2. 关掉记事本
3. 回到黑色窗口，按任意键继续

向导会自动检查格式，显示"✅ 密钥配置完成！"表示填写正确。

> 这一步只做一次，以后直接启动不需要再填。

## 第四步：首次启动

**双击这个文件：**

```
一键启动_Python版.bat
```

### 可能遇到的第一道关卡：Windows 安全警告

Windows 可能弹出蓝色对话框：

```
┌──────────────────────────────────────┐
│  Windows 保护了你的电脑              │
│                                      │
│  Microsoft Defender SmartScreen 阻止 │
│  了一个无法识别的应用...              │
│                                      │
│               [确定]                 │
└──────────────────────────────────────┘
```

**处理方法：**
1. 点击"**更多信息**"（蓝色小字，在正文下方）
2. 出现"**仍要运行**"按钮，点击它
3. 继续正常运行

---

### 启动过程会经历三个阶段

**阶段一：检查环境（约10秒）**

```
  Python 3.x.x OK
  uv OK
  前端已预编译 OK
  检查端口 8000...
```

**阶段二：首次下载依赖（仅第一次，约5~15分钟）**

如果你看到这个框，说明正在下载：

```
  ╔══════════════════════════════════════════════╗
  ║  首次启动：需要下载 Python 依赖（约250MB）   ║
  ║  请耐心等待 5~15 分钟，不要关闭此窗口！     ║
  ╚══════════════════════════════════════════════╝
```

**这期间窗口可能会安静很久，这是正常的，不是卡死了。** 只要窗口还在，就让它跑。

下载需要联网，如果你在公司网络下载失败，可以试试切换手机热点。

**阶段三：启动成功**

看到这行就代表成功了：

```
  ✅ CangJie FOS 已启动
  访问: http://localhost:8000
```

几秒后浏览器会自动打开系统主页。

> ⚠️ **黑色窗口不要关！** 关了系统就停了。  
> 用完系统后再关窗口，或在窗口里按 `Ctrl+C` 停止服务。

---

## 第五步：开始使用

浏览器打开后，你会看到仓颉 FOS 主页。

### 上传路演录音

1. 点右上角「**复盘上传向导**」按钮
2. 填写内容：
   - 投资机构名称（如"红杉中国"）
   - 业务场景（如"01_机构路演"）
   - 被访谈人姓名
3. 点「选择文件」上传录音（支持 `.m4a .mp3 .mp4 .wav`）
4. 点「**提交开始分析**」
5. 等待 **3~10 分钟**（取决于录音时长）
6. 在「**任务列表**」里查看结果

### 问豆豆问题

主页右侧有豆豆对话框，直接打字问，例如：

- "见红杉要准备什么材料？"
- "刚才那份复盘报告里最严重的风险是什么？"
- "我们目前的融资材料还缺什么？"

## 常见问题

| 问题现象 | 解决方法 |
|---------|---------|
| 弹出"Windows 保护了你的电脑" | 点「更多信息」→「仍要运行」 |
| 黑色窗口一闪就消失 | 右键 bat 文件 → 以管理员身份运行 |
| 首次启动卡在下载不动 | 正常，耐心等 10~15 分钟；如一直失败，切手机热点重试 |
| 浏览器没自动打开 | 手动在浏览器地址栏输入 `localhost:8000` 回车 |
| 显示"Python 未找到" | 重新安装 Python，安装时勾选"Add to PATH" |
| 豆豆说"网络不通" | 双击`填写API密钥_双击我.bat`检查 Key 是否正确 |
| 录音上传后一直转圈 | 检查 `DASHSCOPE_API_KEY` 是否填写 |

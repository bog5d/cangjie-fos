# 仓颉 FOS — 本次更新说明

**版本：0.4.1 | 发布日期：2026-05-11**

---

## 这次更新了什么

### 🔍 路演情报报告（全新）
场景选「01_机构路演」时，系统不再打分，而是像老分析师一样解读整场会议：
- 会议氛围评级（🔥 高度积极 / ✅ 正常推进 / ❄️ 兴趣不足）
- 对方每个问题的「背后真实关切」分析
- 兴趣信号、隐性顾虑、关键原声
- 下一步行动建议（含谁来做、是否已承诺）

### 📋 待跟进行动项（全新）
路演分析完成后，「下一步行动」**自动落盘**，出现在主页清单里：
- 主页向下滚动，点「待跟进行动项」展开
- 每条都可以点「完成 ✓」标记，已完成的自动消失

### 🏛 机构时间线（全新）
机构档案侧边栏新增路演时间线：
- 点击机构名 → 侧边栏 → 往下找「路演时间线」
- 按时间倒序，展示与该机构的所有历次路演记录
- 点击任意条目直接跳转对应审查台

### 🔧 修复
- 场景类型（如 01_机构路演）现在正确写入数据库，不再丢失

## 怎么用新功能（具体步骤）

### 测试路演情报报告（推荐用文字稿，30秒出结果）

1. 新建 `test.txt` 文件，写入：
   ```
   说话人A: 你们的退出路径是什么？
   说话人B: 我们计划3年内科创板上市。
   说话人A: 现在ARR大概多少？
   说话人B: 月ARR约50万，今年目标600万。
   ```
2. 点主页「复盘上传向导」
3. 场景选 **01_机构路演**，被访人填任意名字
4. 上传 `test.txt`，点提交
5. 约 30~60 秒后，点「查看报告」

### 查看待跟进行动项

- 分析完成后，**返回主页**向下滚动
- 看到橙色边框的「待跟进行动项」区块 → 点展开

### 查看机构时间线

- 主页 → 资料库 → 机构档案 → 点击某机构
- 侧边栏右侧，找「路演时间线」小节

## 遇到问题

| 现象 | 解决 |
|------|------|
| 一直转圈不完成 | 检查 `DEEPSEEK_API_KEY` 是否填写 |
| 审查台显示「数据异常」 | 重启后端，再点「查看报告」 |
| 待跟进行动项没出现 | 场景必须选 `01_机构路演`；报告需有下一步行动 |

其他问题 → 运行 `CangJie_FOS\诊断_打不开请运行我.bat`

# 仓颉 FOS — 本次更新说明

**版本：0.5.0 | 发布日期：2026-05-11**

---

## 这次更新了什么

### 🎯 路演分析独立工作流（全新 Phase 7.5）

这是本次最重要的新功能。主页顶部新增「**🎯 路演分析**」按钮，点击后进入**独立的5步向导**：

| 步骤 | 做什么 |
|------|--------|
| 步骤1 | 上传录音（mp3/m4a/wav）或**直接粘贴文字稿** |
| 步骤2 | 等待ASR转写（有进度提示；粘贴稿子直接跳过） |
| 步骤3 | **说话人身份确认** — AI预推测每人角色，你填写真实姓名/职位/机构 |
| 步骤4 | AI分析进行中（约30~60秒） |
| 步骤5 | **路演情报报告** — 会议温度、核心关注点、兴趣信号、行动项等 |

**与旧版的核心区别**：AI分析时已知道「说话人0是张明（某VC合伙人）」，分析质量大幅提升。  
支持：引荐方填写 / 竞品识别 / 投资时间线信号 / 主导决策人推测。

### 🔑 强制登录（安全加固）
不管后端配没配账号，**必须输入密码才能进入系统**。
发版时联系王波获取你专属的账号密码。

### 🏛 机构路演统计（优化）
机构卡片新增「**N次路演 · 最近X天前**」统计，一眼看出哪个机构联系最频繁。

### ⚙️ API密钥设置面板
右上角齿轮图标 → 可在界面填写、测试 DeepSeek / DashScope Key，无需手动改 .env 文件。

## 怎么测试新功能（推荐步骤）

### ✅ 路演分析 — 文字稿模式（最快，约1分钟出结果）

1. **登录系统**（账号密码找王波要）
2. 点主页顶部「**🎯 路演分析**」按钮（紫色，在「复盘上传向导」右边）
3. 选「**📝 粘贴文字稿**」标签，把下面这段复制进去：
   ```
   说话人A：你们的退出路径是什么？计划3年还是5年？
   说话人A：现在ARR大概多少？
   说话人B：我们的产品核心壁垒在于AI技术，客户已覆盖500强企业。
   说话人B：商业模式是SaaS订阅制，月ARR约50万，今年目标600万。
   说话人A：这个赛道IRR回报怎么算？你们投资逻辑是什么？
   说话人B：我们计划3年内科创板上市，融资计划已规划清楚。
   ```
4. 路演日期选今天，点「**开始分析 →**」
5. 步骤3：确认说话人 — AI会猜「说话人A → GP执行」「说话人B → 企业方创始人」，填一下姓名
6. 点「**确认身份，开始AI分析 →**」
7. 等30~60秒，步骤5显示「路演情报报告」：
   - 会议温度（🔥/☀️/🧊）
   - 核心关注点
   - 兴趣信号
   - 约定行动项

### ✅ 回归测试（确认旧功能没坏）

- 「**复盘上传向导**」按钮仍正常工作（蓝色按钮）
- 场景选「02_尽调」上传txt → 出现评分界面（有总分和风险点）
- 豆豆对话正常回复

## 遇到问题

| 现象 | 解决 |
|------|------|
| 登录失败 | 联系王波确认账号密码；账号格式区分大小写 |
| 路演分析卡在步骤4不动 | 检查 `DEEPSEEK_API_KEY` 是否填写（右上角 ⚙️） |
| 步骤3没有说话人 | 文字稿格式要有「说话人A:」前缀 |
| 一直转圈 | 点右上角 ⚙️ → 测试密钥是否有效 |
| 音频上传失败 | 先用文字稿模式测试；音频需要 DashScope Key |

其他问题 → 运行 `CangJie_FOS\诊断_打不开请运行我.bat` 或截图发给王波。
